# Content Guidelines

## Medication Language
Never describe the program as explicitly "GLP-1" in marketing copy. The program uses **FDA-approved weight loss medications** which _include_ GLP-1s, dual agonists, and emerging therapies. Framing should always be broader:

- Use: "FDA-approved weight loss medications", "clinically proven medications", "medical weight management"
- Avoid: "GLP-1 program", "GLP-1 weight management", leading with "GLP-1" as the product
- Exception: The Science page and blog posts may reference GLP-1 in educational/explanatory context (e.g., "medications like GLP-1 receptor agonists")
- Exception: SEO metadata keywords may include "GLP-1" since patients search for it, but descriptions should frame it as one option among several

## Image Style
Blog and content images use a **mix of lifestyle editorial and abstract/conceptual** styles. Generated via Grok (XAI_API_KEY in .env, model `grok-imagine-image-pro`).

- **Lifestyle editorial**: Warm, aspirational, human-centric photography. Natural light, soft focus, wellness magazine aesthetic. Use for patient-facing/journey topics (eligibility, lifestyle, results).
- **Abstract/conceptual**: Navy/gold/sage watercolor organic shapes and botanicals. Premium, painterly, matches the site's color palette. Use for scientific/medication topics.
- **Never use**: Marble surfaces, fake injection pens that look like fountain pens, measuring tapes, generic stock photo aesthetics, cold clinical lighting.
- Warm natural tones (cream, warm wood) preferred over cool/sterile surfaces.
